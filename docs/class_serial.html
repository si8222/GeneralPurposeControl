◆ scanMessage()

bool Serial::scanMessage ( std::string &  buffer)
private

Reads the buffer and returns true if a complete message is found.

If so: removes it from the buffer and starts the processing of this message. Determines complete message by waiting for terminator char. This function should be reevaluated for different terminator chars for sending and receiving.

◆ sendScpiCommand

void Serial::sendScpiCommand ( QString  command)
slot

Sends a QString via open serial port.

This slot has to be connected outside of this class. It will append a 'terminator' char at the end of each sent message. (LF : linefeed per default). Terminator char can be selected in the constructor of serial.
